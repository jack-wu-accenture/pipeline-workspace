## Summary

Novartis 2025 MDM Planning Report evaluated four solution paths for replacing the EBX platform. This is the primary reuse artifact for this engagement (REQ-008).

## Four Solution Candidates

| Option | Summary |
|---|---|
| **Reltio** | SaaS MDM; Data Loader & export with customized App; excluded due to global funding constraints |
| **AWS Custom** | MDM UI + Core Modules + Common Services + Data Exchange; custom-built on AWS China |
| **Local Vendor (Sunway)** | ESB localized deployment on AWS China; supports Chinese UI, local business rules, Great Firewall compliant |
| **SFDC** | Salesforce-based; application + technical architecture evaluated |

- Application Architecture Framework

- Technological Selection Evaluation Framework

## Comparison Metrics

- Business Fulfillment
- Maturity Evaluation (Gartner)
- Implementation & License Cost
- Timeline & Key milestones

## Current Status

Reltio was the original recommendation but is now blocked (no global funding). This engagement must re-evaluate AWS Custom vs Local Vendor (Sunway or equivalent) as the viable paths.
